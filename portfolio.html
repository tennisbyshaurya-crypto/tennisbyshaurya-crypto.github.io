cryOS — Intelligent Baby Nursery OS

JAMHacks 10 • University of Waterloo, 2025
React TypeScript Python FastAPI scikit-learn Librosa
Problem: Parents struggle to understand what a baby's cry means — is it hunger, pain, or discomfort?

Solution: I built cryOS, a full-stack application that classifies infant cries in real time using hardware microphone audio. The app uses a Random Forest ML model trained on audio data, processes features with Librosa, and delivers instant classification via a React dashboard.

Results: Achieved 98% accuracy on a held-out test set. Built and demoed the complete product in 24 hours at JAMHacks 10.
